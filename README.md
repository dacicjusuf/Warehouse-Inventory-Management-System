# Warehouse Inventory Management System

This is a Java desktop application for managing products and warehouse inventory.

# Features
- Add, delete, and view products
- Track quantities and calculate total stock value
- Categorize products into Food and Technical categories
- Find products with highest stock value
- Analyze inventory by category and filter by value thresholds

# Technologies
- Java (JVM)
- Collections (Map, List)
- Interfaces and Inheritance

# How to Run
1. Compile all `.java` files in the `src` folder.
2. Run `Main.java`.
